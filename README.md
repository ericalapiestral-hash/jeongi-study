# ⚡ 전기산업기사 필기 스터디

전기 지식이 하나도 없는 상태에서 시작해, **게임처럼 배우면서 문제를 푸는** 전기산업기사 필기 학습 앱입니다.
설치 없이 브라우저에서 바로 돌아가고, 인터넷이 끊겨도 공부할 수 있습니다.

## 무엇이 들어있나

| 구성 | 내용 |
|---|---|
| 🗺 **여정 모드** | 전기 첫걸음부터 시험 5과목까지, 28개 스테이지·129단계를 한 걸음씩. 개념을 이야기로 배우고 → 바로 문제로 확인 |
| 📖 **이론 레슨** | 129개 개념을 만화처럼 말풍선 하나씩 넘기며 학습 |
| ✏️ **문제은행** | 258문제 (본문제 + 쌍둥이 확인문제). 틀리면 쌍둥이 문제로 다시 확인 |
| 📐 **공식 도감** | 공식 31개 — 기호의 뜻·직관·암기법·단계별 예제·변형 공식 |
| ⚡ **계산 도장** | 답이 아니라 풀이를 조립: 공식 고르기 → 값 대입 → 숫자 계산 → 지수 계산 |
| 🎮 **미니 게임** | 무한 등반 · 소거법 훈련 · 버그 헌트 · 공식 빈칸 조립 · 지수 훈련 |
| ⚔️ **보스 레이드** | 과목별 보스 5마리. 정답이 공격, 콤보로 데미지 증가 |
| 🧪 **전기 실험실** | 슬라이더로 만지는 물리 시뮬레이션 8종 |
| 📌 **오답노트** | 틀린 이유 태그 → 약점 레이더 → 맞춤 처방 |
| 🎯 **학습 관리** | 간격 반복(SRS) 복습 · 일일 퀘스트 · 60점 저금통 · 주간 성적표 |

## 학습 설계

- **확신도 베팅** — 답을 고른 뒤 `확실 / 반반 / 찍음`을 선택합니다. 찍어서 맞힌 문제는 마스터로 인정하지 않고 다시 출제해, 예상 점수가 부풀지 않게 합니다.
- **착각 구역** — "확실하다"고 했는데 틀린 문제는 따로 모아 최우선으로 복습합니다.
- **간격 반복** — 맞힌 개념도 1·3·7·14·30일 간격으로 잊어버릴 때쯤 다시 나옵니다.
- **정서 안전장치** — 연속으로 틀리면 이미 마스터한 쉬운 문제를 끼워넣어 세션을 정답으로 마무리합니다.

## 사용법

**웹에서 바로**: [https://ericalapiestral-hash.github.io/jeongi-study/](https://ericalapiestral-hash.github.io/jeongi-study/)

**폰에 설치**: 위 주소 접속 → 크롬 메뉴(⋮) → **홈 화면에 추가** → 앱 아이콘으로 실행 (오프라인 동작)

**PC에서**: 저장소를 받아 `index.html`을 더블클릭

**집 안 네트워크로만 쓰기**: `폰으로 공부하기.bat` 실행 후 표시되는 주소로 접속

## 기록 보관

공부 기록(진도·오답노트·XP)은 서버로 전송되지 않고 **기기의 브라우저에만** 저장됩니다. 기기마다 기록이 따로 쌓입니다.

## 구성

순수 HTML/CSS/JavaScript. 빌드 도구·프레임워크·외부 라이브러리 없음.

```
index.html      화면 골격 · 스크립트 로딩
style.css       전체 스타일 (모바일 반응형)
app.js          핵심 로직 (문제 풀이 · SRS · 통계 · XP)
journey.js      여정 모드 (커리큘럼 · 노드 진행)
data.js         문제은행        lessons.js   이론 레슨
formulas.js     공식 도감       dojo-data.js 계산 도장 문제
dojo.js  more.js  fun.js  boss.js  codex.js  lab.js
sw.js           오프라인 캐시   manifest.json  앱 설치 정보
```

## 문제 정확성

모든 문제는 출제 후 별도의 검증 과정을 거쳐 정답을 교차 확인했습니다. 계산 문제의 최종 답은 수식으로 재계산해 검산했습니다.

> 참고: 전기산업기사 필기는 2021년 CBT 전환 이후 기출문제가 공개되지 않습니다. 이 앱의 문제는 기출을 그대로 옮긴 것이 아니라, 빈출 유형을 분석해 같은 수준으로 새로 만든 오리지널 문제입니다. 전기설비기술기준은 KEC(2021~) 기준을 따릅니다.
